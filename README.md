# kloud.city



## up and running

### required
- node, npm (using 14.15.1)
- vue cli

### install
- cd into directory
- `npm i`

### run
- may need to run `Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass`
- `npm run serve`
- or
- `vue ui`
- import project with project manager
- within vue ui, go to tasks, run `serve` task
